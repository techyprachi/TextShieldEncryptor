2. RSA KEY PAIR

ALGORITHM INTEL

AES

A **symmetric** key algorithm. Fast, efficient, and the global standard for bulk data encryption. Uses a single key for locking and unlocking data.

DES

An older **symmetric** standard. Now considered legacy and vulnerable due to its short key length. Included for educational & historical context.

RSA

An **asymmetric** key algorithm. Uses a public key to encrypt and a private key to decrypt. Slower, but vital for secure key exchange and digital signatures.
